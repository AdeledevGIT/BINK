Deployment Instructions

If the favicon is not showing on Vercel:

  1. Check file deployment: Ensure logo.png is in the root directory
  2. Clear browser cache: Hard refresh (Ctrl+F5 or Cmd+Shift+R)
  3. Check Vercel logs: Look for any 404 errors for logo.png
  4. Verify file size: Ensure logo.png is not too large (< 1MB recommended)
  5. Test different browsers: Some browsers cache favicons aggressively
